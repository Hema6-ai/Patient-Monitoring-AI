# Patient Monitoring AI

A machine learning + deep learning based system that monitors patient vitals in real-time using computer vision and predictive models.

## Features
- Real-time video-based vitals estimation
- Posture detection (fall, lying, sitting)
- Heart rate estimation using facial landmarks
- Oxygen level prediction model
- Alerts for abnormal conditions
- Complete Streamlit dashboard

## Tech Stack
- Python
- TensorFlow / PyTorch
- MediaPipe
- OpenCV
- Scikit-Learn
- Streamlit

More files will be added as we build the system step by step.
